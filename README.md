Lux Bot is a Discord-integrated application designed to manage tournament payments and transactions for online gaming events. The system combines a Discord bot frontend with a Flask-based REST API backend, enabling secure payment processing through Stripe, transaction tracking via Supabase, and automated validation using webhooks.

-Stripe webhook validation for payment confirmation
