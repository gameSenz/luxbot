Lux Bot is a Discord-integrated application designed to manage tournament payments and transactions for online gaming events. The system combines a Discord bot frontend with a Flask-based REST API backend, enabling secure payment processing through Stripe, transaction tracking via MySQL, and automated validation using webhooks.

This project demonstrates backend system design, API development, database integration, and third-party payment processing.

Tech Stack-

**Languages:** Python, SQL

**Backend Framework:** Flask

**APIs:** REST APIs, Stripe API

**Database:** MySQL

**Platform:** Discord

**Tools & Concepts:** Webhooks, Payment Processing, Backend Services, Git, JSON

**System Architecture:**

Discord Bot (Frontend)
        
Flask REST API (Backend)
        
Stripe API & Webhooks
        
MySQL Transaction Database

**Features:**

-Discord bot interface to initiate tournament payment flows

-Secure Stripe Checkout session creation

-Stripe webhook validation for payment confirmation

-MySQL database to store user, tournament, and transaction data

-REST API endpoints for payment verification and payout processing
